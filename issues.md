## 4Ô∏è‚É£ **Options Considered for Markov**

## 5Ô∏è‚É£ **Moving to Hidden Markov Model: New Challenges**

HMM needs:

* Observed sequence ‚Üí from `CPUUtilization`
* Hidden states ‚Üí learned latent process

Initial runs gave:

‚úÖ **emission matrix:** each hidden state emits only Idle with 100%

‚úÖ **transition matrix:** hidden states barely transition ‚Üí stuck in one state

‚úÖ **Viterbi:** hidden path flat

‚Üí why?

üëâ **because observed data = 0 for 50% rows; other 50% saturate at ‚â•100**

üëâ HMM sees **no variability ‚Üí learns trivial model mapping everything to Idle emission.**

---

## 6Ô∏è‚É£ **Diagnosis: Root Cause**

 **Data sparsity in active states** :

‚úÖ too many rows idle

‚úÖ when not idle ‚Üí jumps directly to max

‚Üí no smooth variation in observed `CPUUtilization` ‚Üí impossible for HMM to learn meaningful emission/transition probabilities.
